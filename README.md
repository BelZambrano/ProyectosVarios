# ⏰ Recordatorio de Click – (insertar link) 

Este proyecto nació desde una necesidad real:  
cuando estoy en clases o programando, muchas veces se me pasa el tiempo y la plataforma de SENCE cierra la sesión por inactividad.  
Así que decidí crear una herramienta simple que me avise a tiempo para no perder la conexión.

---

## ¿Qué hace esta página?

Es un recordatorio visual y sonoro que:

- Muestra un contador regresivo  
- Lanza una alerta cuando se acaba el tiempo  
- Reproduce un sonido tipo notificación  
- Puede abrir automáticamente la página donde tengo que hacer click  
- No deja ignorar la alerta fácilmente  

La idea es que funcione como un “empujoncito” cuando estoy concentrada en otras cosas.

---

## ¿Cómo se usa?

1. Abres el archivo  
recordatoriosimple.html

2. Configuras:
- Cada cuántos minutos quieres el aviso  
- El link de la página (por ejemplo, Aula Digital SENCE)

3. Presionas **Iniciar**  
Y listo. La página se encarga del resto.

---

## Cuando aparece la alerta

La ventana:

- Suena con un “pip-pip”  
- Intenta abrir la página  
- No se cierra con Escape  
- No se puede cerrar haciendo click afuera  

Solo se cierra con **Listo** o presionando **Enter**.

Esto lo hice así a propósito, para no ignorarla sin darme cuenta.

---

## Botones principales

- **Iniciar** → Comienza el contador  
- **Pausar** → Detiene el conteo  
- **Reiniciar** → Vuelve al tiempo inicial  
- **Abrir link** → Abre la página manualmente  
- **Probar alerta** → Muestra una alerta de prueba  
- **Borrar ajustes** → Borra la configuración guardada  

---

## Sonido

El aviso usa un sonido tipo notificación (pip-pip).  
Para que el navegador permita reproducirlo automáticamente, primero hay que presionar **Probar alerta** una vez.

---

Porque:

Esta página me ayuda a organizarme mejor sin estar mirando el reloj todo el rato.
